CSET-110 Lab:
Project Structure

Today, you're going to build your first project!

We are going to use our plans from yesterday and turn our site into a reality. Go through the lecture slides from yesterday and today again if you need to, but by now you should have everything you need to dive in. This is just like the lab from the first week, except we're making a few more pages. It's up to you to get everything working together properly.

When you're done, you're going to zip your entire project into one file and upload it to the portal. If you don't know how to do that, read this article. There's even a way to do this in the command line, for those that are curious!

Rubric

  • Create a project with a well organized directory structure.
  • Create at least 2 child directories.
  • Create at least 5 different pages
    • All pages need to be valid HTML documents.
    • Your home page is the only page that needs to contain the structural elements that we learned yesterday.
    • Other pages must contain some content specific to that page, like the page's heading and a paragraph describing why the page exists.
    • You need to create at least 2 index pages.
  • Use at least 2 local images. (We aren't going to use other media files so we can save on size.)

Tips

  • Refer back to the article we read yesterday if you need any help with your structural elements.
  • Don't worry about styles! You can use comments in your code if you want to remember how you plan on visually representing your content, but this lab is all about using the proper semantic structure of your pages. Focus on that.
  • Make sure you test all of your links to see that they take you where you expect.
  • If you want to fill out your page with extra dummy content and don't want to write it by hand, you can use a Lorem Ipsum generator to add some generic copy. Remember, I still want to see some real content on every page so I know why the page exists, this is just to help make it look less barebones.
  • And have fun with it! I don't care what your content is, I care that you care what it is. :)
