CSET-110:
More CSS

Today, we're going to learn more details about CSS.

In particular, we'll discuss how different styles are applied if there are conflicting rules using CSS cascade model, rule inheritance, and selector specificity. We'll dive deeper into selectors in order to target more complex patterns of elements. And then we'll explore the Box Model to start learning properties related to page layouts.
