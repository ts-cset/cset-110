Deployment Resources

  • Using GitHub Pages - Instructions for deploying static HTML sites using GitHub's service. Make sure you select the correct type of project: User Site or Project Site.
  • Anatomy of a URL - Helpful blog post to explain the parts of a URL.
  • MDN: Publishing your website - Different strategies for finding hosting and domains and using FTP programs. Also discusses other online publishing platforms like GitHub Pages.
  • DigitalOcean Tutorials - DO is a hosting service that has excellent tutorials on setting up your own server, configuring web server software, and other helpful tools. Many of these instructions will work on other hosting services, too.
  • You'll also need a domain name. Follow the instructions that your hosting provider and domain registrar offer to connect the two. For example, here's DigitalOcean's guide.
